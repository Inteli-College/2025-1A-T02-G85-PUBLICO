# Public Report Module 3

## Sprint 1

### Workload

This sprint focused on implementing authentication and authorization mechanisms to secure user access within the platform. Additionally, the team refined the system workflows to ensure a smoother and more efficient user experience. These improvements established the foundation for controlled access to the platform and better alignment between user permissions and functionalities.

### Learning

- Authentication and authorization concepts  
- Security best practices in web applications  
- Workflow optimization for system efficiency

### Workload

### Workload

The final sprint of Module 3 centered on deploying the platform into a cloud environment and integrating continuous deployment (CD) processes. ### Learning
